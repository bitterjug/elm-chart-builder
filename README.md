# elm-chart-builder
An opinionated and accessible charting library for Elm based on [gampleman/elm-visualization](https://github.com/gampleman/elm-visualization).

## examples:
`elm reactor`

Then navigate to the examples folder in the browser.

## tests:
`elm-test`
